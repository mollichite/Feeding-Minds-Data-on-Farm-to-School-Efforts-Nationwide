## Data Sources


- United States Department of Agriculture 2023 Farm to School Census 
    https://www.fns.usda.gov/research/f2s/census2023
    
    *This dataset was paired down from 930 to 120 columns.  The edited data file is provided in the repository.  The raw data can be downloaded from the link provided.

    https://nces.ed.gov.qipservices.com/ccd/elsi/


## Getting Started
    1. Fork and clone this repository
    2. Create and activate a virtual environment

        windows:\
        python -m venv venv\
        source venv/Scripts/activate 

        mac/linux:\
        python3 -m venv venv\
        source venv/bin/activate

    3. Install requirements.txt file
        pip install - r requirements.txt
